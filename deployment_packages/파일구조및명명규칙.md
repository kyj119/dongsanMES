## 📝 파일명 규칙

### 기본 형식
```
[카드번호]_[설명].eps

예시:
20260206-03-2_달력.eps
│      │ │ │ └─ 설명 (선택)
│      │ │ └─── 품목 순번 (1, 2, 3, ...)
│      │ └───── 주문서 순번 (01, 02, 03, ...)
└──────┴─────── 날짜 (YYYYMMDD)
```

### 카드번호 구성
```
YYYYMMDD-XX-Y
│      │ │ │ └─ 품목 순번 (Item Number)
│      │ │ └─── 주문서 순번 (Order Sequence)
└──────┴─────── 작성일 (Date)

예시:
- 20260206-01-1 : 2026년 2월 6일, 첫 번째 주문서, 첫 번째 품목
- 20260206-01-2 : 2026년 2월 6일, 첫 번째 주문서, 두 번째 품목
- 20260206-02-1 : 2026년 2월 6일, 두 번째 주문서, 첫 번째 품목
```
